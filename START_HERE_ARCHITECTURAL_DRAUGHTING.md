# Start Here: Architectural Draughting Students

Welcome to the Synergy Inc. Architectural Draughting repository.

You will use this repository to learn how project research, layout concepts, draughting exports, feedback, and revisions can be managed professionally using GitHub.

## Your Main Focus

Your work will focus on the Synergy Inc. project, a mixed-use development that may include offices, restaurants, doctors' rooms, mechanical workshops, a diamond cutting house, a gold refinery, a jewellers hub, and an auction house.

## What You Need To Prepare

Please prepare:

- A GitHub account
- Access to your preferred draughting/Draughting software
- A folder on your computer for internship files
- A notebook or digital notes file
- A basic understanding of PDF export from your draughting tool
- Your weekly availability and exam/study constraints

## First Week Training Focus

The first week is not about creating final drawings. It is about understanding:

- The project brief
- GitHub workflow
- File naming and version control
- Research expectations
- Zoning and movement-flow concepts
- Public, staff-only, service, and restricted areas
- How to export work for review
- How to respond to feedback

## Communication Rule

Because the internship is remote, communicate early if you are stuck, unavailable, or unsure what to do next.
