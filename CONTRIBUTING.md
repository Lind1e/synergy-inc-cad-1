# Contribution Guidelines

## Before You Start Work

1. Read the project brief.
2. Check the latest files and notes.
3. Confirm which task you are working on.
4. Do not overwrite another student's work.
5. Use the correct folder.

## When Saving Work

- Use the correct naming convention.
- Export a PDF, PNG, or JPG for review when possible.
- Add short notes explaining your assumptions.
- Update the revision log.
- Commit with a clear message.

## Commit Message Examples

```text
Add office space planning research
Add initial zoning concept v01
Export workshop layout for review
Update parking access concept after feedback
Add movement flow notes for public and restricted areas
```

## Do Not Commit

- Temporary Draughting files
- Duplicate exports
- Personal files
- Unnamed files
- Files called final-final
- Very large source files unless approved
- Confidential or private information
