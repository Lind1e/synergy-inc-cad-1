# First Week Training: Architectural Draughting

The first week is focused on training, research, and preparing students to contribute properly.

## Day 1: Remote Orientation and Project Brief

- Read the welcome and orientation notes.
- Read the Synergy Inc. project brief.
- Confirm your availability and exam/study commitments.
- Confirm your GitHub username.
- Confirm which draughting software you can access.

## Day 2: GitHub and File Organisation

- Learn the repository folder structure.
- Read the naming convention.
- Practise creating a small markdown note.
- Practise committing work to GitHub.
- Learn how exports should be saved for review.

## Day 3: Site Research and Space Planning

Research the main space types:

- Offices
- Restaurants
- Doctors' rooms
- Mechanical workshops
- Diamond cutting house
- Gold refinery
- Jewellers hub
- Auction house

Focus on practical requirements, public access, restricted zones, service access, and movement flow.

## Day 4: Zoning and Movement Flow

- Draft basic zoning ideas.
- Separate public, staff-only, service, and restricted areas.
- Think about vehicle access and parking.
- Prepare a simple diagram or notes for review.

## Day 5: First Starter Deliverable

Submit one of the following:

- A research note for one space type
- A zoning concept sketch/export
- A movement-flow diagram
- A parking/access concept
- A short explanation of assumptions and questions
